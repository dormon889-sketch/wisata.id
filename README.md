# 🗺️ Jelajah Nusantara - Eksplorasi Keajaiban Indonesia

Sebuah platform visual minimalis yang menyajikan kurasi rekomendasi wisata terbaik dari seluruh penjuru Nusantara. 

## 🏔️ Kenapa Website Ini?
- **Immersive Visual:** Menggunakan gambar berkualitas tinggi untuk memberikan kesan nyata.
- **Simpel & Elegan:** Tidak banyak teks yang mengganggu, hanya informasi penting.
- **Responsif:** Nyaman dibuka dari perangkat apapun (Laptop/Tablet/HP).

## 💻 Cara Instalasi
1. Buat folder baru.
2. Masukkan file `index.html` dan `style.css`.
3. Buka `index.html` di browser Anda.
